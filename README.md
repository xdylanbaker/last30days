# last30days

Generate comprehensive reports of your last 30 days across apps and services.

## Examples

```bash
# Generate daily report
openclaw skill last30days generate --apps "discord,github,notion" --format markdown

# Specific time range
openclaw skill last30days range --start "2026-01-08" --end "2026-02-08"
```

## Installation

```bash
openclaw skills install xdylanbaker/last30days
```

## Local Development

```bash
cp -r skills/last30days ~/.local/share/openclaw/skills/
```
